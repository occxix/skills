# 小说章节解析器（知识库自动构建） v2.0

严格按照「三次操作 + 固定树状结构 + 精确模板」处理每一章小说。**每读完一章必须执行三次操作，顺序绝对不可颠倒**。

## 使用方法

### 初始化新小说
```
/init-novel <小说名称>
```
创建 `小说解析器知识库（《小说名称》）` 目录结构。

### 处理单章
```
/parse-chapter <章节号>
```
按三次操作流程处理该章节。

### 批量处理
```
/parse-chapters <起始章> <结束章>
```
依次处理指定范围内的所有章节。

## 核心铁律（不可违反）

- 操作1：**纯提取**（只读本章原文，不查询知识库、不做任何判断），预处理只记录章节起始行号和章节结尾行号
- 操作2：**只查询**（不修改任何文件）
- 操作3：**一次性写完**所有更新（实体当前状态 + 变化日志 + 章节总结）
- 关键记忆点必须写入 `9_关键记忆点/`，包括高能语录、关键内心和思想变化、剧情转折
- 所有文本文件必须自动处理编码问题：读取时识别 UTF-8/GB18030/GBK/Big5 等常见编码并修复常见乱码，写入时统一保存为 UTF-8
- 重大章节（卷末、高潮、重大转折）允许章节总结更详细，但必须严格使用6维度模板
- 全局回顾时优先阅读 `8_章节总结/` 文件夹 + `7_变化日志/总变化时间线.md`
- `8_章节总结/` 负责本章总览；`1_写作手法/`、`4_剧情与事件/`、`5_关系网络/`、`6_动态演变与主题/` 负责长期沉淀，禁止只写章节总结而遗漏长期档案
- `1_写作手法/` 必须每章一份，同时维护 `写作手法总览索引.md` 和 `全书写作手法总结.md` 滚动汇总
- `4_剧情与事件/` 每章判断是否有可沉淀事件；无关键事件可不写入，有事件必须更新对应剧情/事件档案
- `6_动态演变与主题/` 使用“滚动总览 + 章节证据”模式，既更新长期判断，也保留第XX章证据来源
- `5_关系网络/` 必须保持全自动：小变动补充当前版，大变动自动在同一文件内归档历史版本并改版，不向用户请求确认

## 完整三次操作流程（v2.0 精确版）

### 操作1：生成简单预处理

文件路径：`0_预处理/第XX章预处理.md`

**必须使用以下固定模板（纯提取）**：

```markdown
# 第XX章 预处理（纯提取）

**章节基本信息**
- 标题：
- 字数：
- 时间/地点：
- 章节起始行号：第X行
- 章节结尾行号：第Y行

**提取内容**

**人物**
- 人物名：描述（能力、性格、外貌、关键行为）

**组织**
- 组织名：描述（地位、与主角关系）

**地域**
- 地域名：描述（特点、氛围）

**剧情与事件**
- 主事件：
- 关键细节：

**关系**
- A → B：关系描述

**世界观提示**
- 新增/提及设定：

**伏笔/象征**
- 伏笔：
- 象征：

**关键记忆点候选**
- 高能语录：原句/概述
- 关键内心和思想变化：描述
- 剧情转折：描述
```

### 操作2：根据预处理查询当前状态

**只查询、不修改**。必须输出以下固定格式的《查询报告》：

```markdown
# 第XX章 查询报告

**当前知识库状态（查询前）**
- 实体名：状态描述（新增 / 已存在 / 需要更新）

**需要更新的实体**
- 地域：xxx（新增）
- 组织：xxx（新增）
- 人物：xxx（新增/更新）

**需要写入关键记忆点**
- 高能语录：xxx
- 关键内心和思想变化：xxx
- 剧情转折：xxx

**是否需要调用历史时间线？** → 是/否（原因）
**是否需要调用之前章节总结？** → 是/否（原因）
**是否需要调用关键记忆点？** → 是/否（原因）

查询完成，可直接进入操作3更新。
```

### 操作3：完成知识库更新（一次性写完）

**必须按以下顺序一次性输出所有内容**：

#### 3.1 更新实体文件

每个实体文件（人物、地域、组织等）**必须使用以下标准格式**：

```markdown
# 实体名称

## 1. 当前状态
- 首次出现：第XX章
- 身份/类型：
- 特点：
- 能力/设定：
- 最新状态：（本章关键变化）

## 2. 变化时间线
- 第XX章：具体变化描述
```

#### 3.2 写入变化日志

- 在对应分类日志（如 `人物变化日志.md`、`地域变化日志.md` 等）**追加记录**
- 同时在 `7_变化日志/总变化时间线.md` 追加本章总览

#### 3.3 写入长期沉淀档案

**必须先完成长期档案，再生成章节总结。**长期档案是跨章节检索入口，章节总结只是本章快照。

##### 3.3.1 写作手法（每章添加）

文件：
- `1_写作手法/第XX章写作手法.md`
- `1_写作手法/写作手法总览索引.md`
- `1_写作手法/全书写作手法总结.md`

`第XX章写作手法.md` 必须记录本章实际出现的叙事技巧、视角调度、节奏控制、伏笔铺设、象征/意象、对白与动作描写特点，并标注原文章节证据。

`写作手法总览索引.md` 只做索引和定位：按技巧类型列出出现章节，方便回查。

`全书写作手法总结.md` 作为滚动汇总：只在本章新增、强化或反转既有写作模式时更新，不做机械重复。

##### 3.3.2 剧情与事件（分析生成）

每章判断是否存在可沉淀事件。若本章没有推进主线、支线、关键冲突、谜团、伏笔兑现、危机升级或目标变化，可在操作3说明“本章无新增剧情事件档案写入”。

若存在事件，必须按需要更新：
- `4_剧情与事件/主线剧情时间线.md`
- `4_剧情与事件/子剧情分支.md`
- `4_剧情与事件/关键事件记录.md`

写入时必须区分“事实事件”和“分析判断”：事实事件来自原文，分析判断必须标注依据。

##### 3.3.3 关系网络（分析生成与补充）

默认规则：
- 小变动：补充 `当前关系图谱` 和 `变更记录`
- 大变动：自动执行“历史版本归档 + 当前版改版”
- 每次补充必须写明章节来源、关系双方、关系类型、变化原因、证据句/情节依据

大变动判定标准：背叛、结盟、敌友转换、身份揭露、阵营/组织归属变化、权力结构改变、关键人物死亡/失踪、关系公开或决裂、长期误解解除、主从/师徒/盟友等核心关系性质改变。

同一关系文件必须使用以下结构，保持全自动、可读、可追溯：

```markdown
# 关系对象/关系网络名称

## 当前关系图谱
- A ↔ B：当前关系类型、强度、立场、利益绑定、信任状态

## 变更记录
- 第XX章：小变动/大变动；变化内容；原因；证据

## 历史版本归档
### v1（第XX章前）
- 归档原因：
- 当时关系图谱：
- 被第XX章哪些事件改写：
```

自动改版规则：
- 小变动只追加 `变更记录` 并修正 `当前关系图谱`
- 大变动先把改版前的 `当前关系图谱` 复制到 `历史版本归档`，再重写 `当前关系图谱`
- 版本号按 `v1`、`v2`、`v3` 顺序递增
- 证据不足但关系可能变化时，写入 `变更记录` 的“待观察”项，不改版
- 禁止因为关系复杂而询问用户；必须基于原文和知识库自动完成

##### 3.3.4 动态演变与主题（滚动总览 + 章节证据）

文件：
- `6_动态演变与主题/主题演变总览.md`
- `6_动态演变与主题/象征与隐喻记录.md`
- `6_动态演变与主题/核心主题变化轨迹.md`

更新方式：
- 滚动总览：维护当前阶段的主题判断、人物/势力/剧情动态走向
- 章节证据：追加第XX章的具体证据，包含情节、意象、对白、人物选择或叙事结构
- 不确定判断必须标注“待观察”，不得提前定论

#### 3.4 生成章节总结

文件：`8_章节总结/第XX章总结.md`

**必须使用以下6维度完整模板**：

```markdown
# 第XX章 总结

**章节基本信息**
- 章节标题：
- 字数：
- 核心事件：

**1. 写作手法与叙事技巧**

**2. 世界观与环境设定**

**3. 人物体系**

**4. 剧情与事件体系**

**5. 关系网络**

**6. 动态演变与主题**

**本章贡献**：

**待关注**：

**关键记忆点**：
- 高能语录：
- 关键内心和思想变化：
- 剧情转折：

**来源**：第XX章原文 + 知识库查询
```

#### 3.5 写入关键记忆点

- `9_关键记忆点/高能语录记录.md`：记录本章值得复用、体现人物锋芒或主题张力的语录。
- `9_关键记忆点/关键内心和思想变化记录.md`：记录人物认知、信念、目标、价值判断的明显变化。
- `9_关键记忆点/剧情转折记录.md`：记录改变局势、目标、阵营、危机等级或读者预期的转折。

## 知识库树状结构（固定，不可更改）

│   ├── 第01章查询报告.md
│   ├── 第02章预处理.md
│   ├── 第02章查询报告.md
│   ├── 第03章预处理.md
│   ├── 第03章查询报告.md
│   └── ...（后续每章对应一个文件）

├── 1_写作手法/
│   ├── 第01章写作手法.md
│   ├── 第02章写作手法.md
│   ├── 写作手法总览索引.md           ← 按技巧类型索引出现章节
│   └── 全书写作手法总结.md           ← 滚动汇总全书写作模式

├── 2_世界观与环境/
│   ├── 地域库/
│   │   ├── 霜都.md
│   │   ├── 古旧神社.md
│   │   └── ...（每处重要地域一个文件）
│   ├── 组织库/
│   │   ├── 玄影会.md
│   │   ├── 天机阁.md
│   │   └── ...（每处重要组织一个文件）
│   └── 世界观总览.md                  ← 全局世界观设定总览

├── 3_人物体系/

│   ├── 寒霜.md

│   ├── 象征与隐喻记录.md             ← 章节证据
│   └── 核心主题变化轨迹.md           ← 主题判断 + 章节证据

├── 7_变化日志/                        ← 所有变化专属记录
│   ├── 人物变化日志.md
│   ├── 组织变化日志.md
│   ├── 地域变化日志.md
│   ├── 剧情与子剧情变化.md
│   ├── 关系变化日志.md
│   └── 总变化时间线.md                ← 全局时间线总览

├── 8_章节总结/
│   ├── 第01章总结.md
│   ├── 第02章总结.md
│   ├── 第03章总结.md
│   ├── 全书章节总结索引.md           ← 可选：章节总目录+关键点索引
│   └── ...（每章一个总结文件）

└── 9_关键记忆点/                      ← 新增：高价值记忆点专区
    ├── 高能语录记录.md               ← 经典台词、金句、名场面对话
    ├── 关键内心和思想变化记录.md     ← 人物心理转变、内心独白高光
    └── 剧情转折记录.md               ← 重大反转、伏笔兑现、命运节点
