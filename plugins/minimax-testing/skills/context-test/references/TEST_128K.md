## 结论

### 128K 上下文能力排名

1. **M2.7**: 100% 通过，最快 (93字/秒)
2. **Astron**: 100% 通过，稳定但慢 (6字/秒)
3. **M2.5**: 80% 通过，有截断问题 (33字/秒)

### 速度对比

| 模型 | 短文本速度 | 128K速度 |
|------|-----------|----------|
| M2.5 | 300字/秒 | 33字/秒 |
| M2.7 | 180字/秒 | 93字/秒 |
| Astron | 16字/秒 | 6字/秒 |

### 建议

- **长上下文任务 (50K+)** → 使用 M2.7 或 Astron
- **快速生成** → 使用 M2.5
- **代码理解** → 使用 Astron (128K)
- **综合能力** → M2.7 最优
