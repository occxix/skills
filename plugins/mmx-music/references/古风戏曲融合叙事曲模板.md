# 古风戏曲融合叙事曲模板

基于《女者》创作经验总结。

---

## 适用场景

- 古风叙事长曲
- 戏曲念白 + 主声双声部
- 仪式感、情感递进
- 时长 4 分钟左右

---

## 歌词创作规范

### 1. 韵脚规则

- **每段统一韵脚**：同一段落内所有句子押同一韵
- **段落间换韵**：避免重复，按情绪选择韵脚
- **韵脚情绪对照**：

| 情绪 | 推荐韵脚 | 示例字 |
|------|----------|--------|
| 悲凉、空灵 | ang/iang | 央、香、长、芳 |
| 温暖、舒缓 | ou/iu | 秋、留、修、柔 |
| 深情、内敛 | en/an | 颤、人、门、春 |
| 悲壮、决绝 | ing/eng | 姓、名、听、轻 |
| 沉郁、厚重 | u/ü | 谱、户、顾、误 |

### 2. 句式结构

- **七言为主**：情感丰富，适合副歌、桥段
- **五言为辅**：节奏轻快，适合主歌
- **四言点缀**：古朴庄重，适合引子、念白

### 3. 念白腔处理

- 念白腔独立成段，不打断主声韵律
- 使用 `——` 标注拖腔
- 示例：
  ```
  （戏腔）
  吉——日——令——辰——
  始——加——元——服——
  ```

## 结构模板

```
[Intro]        引子（戏腔念白或器乐）
[Inst]         器乐前奏
[Verse]        主歌 1：叙事开场
[Verse]        主歌 2：情节发展
[Inst]         器乐间奏
[Pre Chorus]   预副歌：情感转折
[Verse]        主歌 3：深化
[Inst]         器乐间奏
[Build Up]     积累段：高潮前铺垫
[Chorus]       副歌：情感高潮
[Inst]         器乐间奏
[Solo]         独奏段
[Bridge]       桥段：情绪爆发
[Inst]         器乐间奏
[Outro]        终章
[Outro]        尾声（镜像或重复）
```

## Prompt 模板

### 基础模板

```
中国古风戏曲融合叙事长曲，极慢板 BPM 60，D minor 小调悲凉空灵。双声部——念白腔（清冷戏腔极慢拖腔）与主声（温柔女声平静陈述）对话。情绪层层递进，整曲舒展宏大不少于四分钟。引子编钟+洞箫极慢空灵，主歌古筝轻拨，仪式段二胡入场情感转折，副歌人声全奏器乐退后，器乐间奏二胡+古筝独奏，入族谱段戏腔独立成段一字一顿，终章归于寂灭。
```

### 参数说明

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| `--model` | music-2.6 | 付费版，质量更高 |
| `--bpm` | 60 | 极慢板，空灵意境 |
| `--key` | D minor | 小调悲凉（模型可能忽略） |
| `--vocals` | 双声部描述 | 念白腔 + 主声 |
| `--instruments` | 编钟, 洞箫, 二胡, 古筝, 琵琶, 弦乐 | 古风标配 |
| `--genre` | 古风戏曲融合 | 流派 |
| `--mood` | 悲凉, 空灵, 内敛深情 | 情绪标签 |

## 完整示例：《女者》

### 歌词

```
[Intro]
（戏腔念白，极慢）
女——者——
从——人——者——也——

[Inst]
（洞箫独奏，空灵）

[Verse]
父亲书房夜未央
灯花燃尽又添香
天明为我取名字
唤作兰儿意未长

[Verse]
庭中兰草年年芳
母亲教我敛容光
低头沏茶学妇道
不敢抬头看四方

[Inst]
（古筝轻拨）

[Pre Chorus]
（戏腔）
吉——日——令——辰——
始——加——元——服——

（主声）
母亲簪钗手未颤
说从今日是大人
我低头应声归去
不知归处是何门

[Verse]
玉兰花开几度秋
抄完女诫墨痕留
贞妇不留名姓在
消失方是功德修

[Inst]
（二胡入场，悲凉）

[Build Up]
（戏腔）
凤——冠——霞——帔——
十——里——红——妆——

（主声）
上轿前父亲立廊下
轿帘落时见背影
一盆清水泼出去
从此不是家中人

[Chorus]
一拜天地冠夫姓
二拜高堂谢养育
夫妻对拜抬头看
第一次见他的脸

诗里那人名有姓
他可知道我是谁
我本兰儿唤作招娣
如今只剩某氏名

[Inst]
（二胡独奏，情感高潮）

[Solo]
（古筝+二胡交织）

[Bridge]
（戏腔，一字一顿）
某——氏——
讳——招——娣——
适——某——门——

（主声）
红纸族谱写几行
我在其中第几章
本名附录存其迹
不过寥寥两字长

[Inst]
（编钟+洞箫，归于寂灭）

[Outro]
世间招娣千千万
悲喜皆循礼教传
从来无人问心意
我也识得字如山

[Outro]
（极轻，渐弱）
我也识得字如山
```

## 实测数据

| 版本 | 时长 | BPM | 调性 | 频谱中心 |
|------|------|-----|------|----------|
| v1 | 2:49 | 120.2 | D major | 2348 Hz |
| v2 | 3:46 | 60.8 | D major | 1985 Hz |
| v3 | 3:53 | 90.7 | D major | 1814 Hz |

### 结论

- **BPM**：模型可能加快，设定 60 实际 60-90
- **调性**：模型忽略小调设定，始终大调
- **时长**：增加 [Inst] 可延长时长
- **频谱中心**：越低越沉郁

---

## 注意事项

1. **BPM 无法精确控制**：模型会根据歌词长度自动调整
2. **调性无法强制**：MiniMax 模型限制，始终偏向大调
3. **双声部效果有限**：模型对"双声部"理解有限，可能混在一起
4. **副歌能量**：需在 prompt 强调"副歌人声全奏器乐退后"
5. **归档旧版本**：重新生成前先归档，避免覆盖
