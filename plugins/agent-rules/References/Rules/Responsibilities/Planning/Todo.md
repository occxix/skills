# Todo Planning Rules

Use this file when turning an analyzed request into actionable tasks.

## Core Rules

- Break work into ordered, independently actionable tasks.
- Keep tasks small enough to verify.
- Put dependencies before dependent work.
- Mark tasks that need user confirmation or external input.
- Keep task wording concrete and implementation-oriented.

## Output

- Ordered task list
- Dependencies
- Risks or blockers
- Suggested execution order
